Avatar showing an image or initials fallback. Used for teacher profiles and tool-author bylines.

```jsx
<Avatar name="Maria Santos" />
<Avatar name="Pat Lee" src="/img/pat.jpg" size="lg" />
<Avatar name="AI Tools" shape="square" tone="clay" />
```

Sizes: `xs · sm · md · lg`. `shape`: circle/square. `tone`: pine/clay/ochre.
