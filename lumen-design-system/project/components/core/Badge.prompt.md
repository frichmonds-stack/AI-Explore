Small categorical label for counts, categories and metadata. (For a tool's CEWA approval state, reach for `StatusBadge` instead.)

```jsx
<Badge tone="primary">Featured</Badge>
<Badge tone="ochre" dot>Popular</Badge>
<Badge tone="neutral" variant="outline">12 tools</Badge>
<Badge tone="accent" variant="solid">New</Badge>
```

Tones: `neutral · primary · accent · ochre · success · warning · danger · info`. Variants: `soft` (default) · `solid` · `outline`. Add `dot` for a leading status dot.
