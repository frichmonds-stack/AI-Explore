Base surface container — warm white, 16px radius, soft warm shadow. Pass `interactive` for whole-card link behaviour (hover lift).

```jsx
<Card pad="md">…</Card>
<Card interactive as="a" href="#" pad="lg">…</Card>
<Card elevation="flat" pad="sm">…</Card>
```

Props: `pad` (none/sm/md/lg) · `elevation` (flat/sm/raised) · `interactive` · `as`.
