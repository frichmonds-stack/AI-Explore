Pill-shaped action button — use for any primary or secondary action; `primary` (pine) is the default CTA, `accent` (clay) for warm emphasis.

```jsx
<Button variant="primary" size="md">Browse AI tools</Button>
<Button variant="accent">Suggest a tool</Button>
<Button variant="secondary" size="sm">Filters</Button>
<Button variant="ghost">Learn more</Button>
<Button as="a" href="#" variant="primary" block>Get started</Button>
```

Variants: `primary` · `accent` · `secondary` (outline) · `ghost`. Sizes: `sm` · `md` · `lg`. Props: `block`, `disabled`, `as`. Hover lifts 1px with a pine-tinted shadow; never bouncy.
